## Usage

`` `sh
> git clone https://github.com/kai-kou/use-cognito-totp-mfa.git
> cd use-cognito-totp-mfa
> npm install

# Set Cognito user pool ID and client application ID
> vi config.js.example

> node mfa-auth.js
`` `
